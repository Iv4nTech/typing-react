# Typing React

Es una aplicación para practicar mecanografía, donde puedes registrarte y empezar a escribir. 

### Características
- **Frases aleatorias**: Las frases que aparecen para practicar son generadas al azar.
- **Sin presión**: No hay límite de tiempo, ideal para practicar a tu ritmo.
- **Seguimiento**: Cada frase que completas se guarda automáticamente como una sesión completada.

## Tecnologías de Estilado

He elegido **CSS puro** para estilizar la aplicación. Para mantener un código limpio y organizado, cada componente tiene su propio archivo CSS dedicado.
